How to build
============

**It should be built** from the ``mosaic-distribution`` repository, as it depends on a proper set workbench and symlinks to other
repositories...

* dependencies:

 * ``nodejs`` version ``0.8.x``;
 * ``npm`` latest version;
 * correct setup of symlinks towards other source code repositories;

* inside a console, into the root folder of the project: ::

  ./scripts/prepare
  ./scripts/compile

How to run
==========

* inside a console, into the root folder of the project: ::

  ./scripts/run-service


How to use
==========

You'll need to start a ``mosaic-node`` instance.

Point your browser at http://127.0.155.0:31804/ and you'll get access to an administrative console.
